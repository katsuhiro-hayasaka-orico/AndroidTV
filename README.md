# Minimal TV Clock

Android TVで常時表示する、ローカルWebViewベースのミニマルな時計・カレンダーアプリです。

iPhoneの横向きスタンバイ表示を参考に、情報量を増やさず、遠距離でも読みやすい時計・日付・当月カレンダーだけを表示します。

## 表示内容

- 現在時刻（HH:mm、秒なし）
- デジタル時計 / アナログ時計の切り替え表示
- 日付と曜日
- JavaScriptで自動生成する当月カレンダー
- 今日の日付の強調表示

外部API、外部カレンダー、予定データ、社内情報、外部Webフォントは使用しません。

## 操作方法

- 初期表示はデジタル時計です。
- Android TVリモコンの決定ボタン、またはキーボードの Enter キーで、デジタル時計とアナログ時計を切り替えます。
- 選択した時計表示モードはWebViewの `localStorage` に保存され、次回起動時も維持されます。
- 戻るボタンでアプリを終了します。

## デザイン方針

- 黒単色ではなく、濃紺・黒鉛・チャコール系の仄かなグラデーション背景を使用します。
- カレンダーは控えめな半透明のガラス調パネルにし、固定ロゴや不要な装飾は追加しません。
- アナログ時計は秒針を省き、時針と分針だけの薄いガラス調・ラインアート調の盤面にしています。
- デジタル時計はローカル環境で利用可能な幾何学的サンセリフ系フォントを優先し、`font-variant-numeric: tabular-nums` を維持します。
- 日本語の日付・曜日・カレンダーは崩れにくいシステムフォントスタックを使用し、3〜5m離れても読めるサイズを保ちます。
- 常時表示対策として、既存の `nudgeDisplay` による微小位置移動を維持し、秒単位で動く装飾や過度なアニメーションは入れていません。

## Android TV対応

- `LEANBACK_LAUNCHER` でTVホーム画面から起動できます。
- 横向き全画面表示を前提にしています。
- 戻るボタンでアプリを終了します。
- Activity側でスリープ抑止を設定しています。
- WebView側でネットワークロードをブロックし、ローカルアセットのみを表示します。

## ビルド

Android Studioでこのディレクトリを開き、Android SDK / Gradle Pluginを同期してから実行してください。

```bash
gradle assembleDebug
```
